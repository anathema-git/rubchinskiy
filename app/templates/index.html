О системе

Эта система реализует алгоритм справедливого дележа между двумя участниками (A и B). Алгоритм позволяет найти пропорциональный делёж, при котором каждый участник получает не менее половины от своей оценки всех предметов.

Подробнее об алгоритме
  1. Построение ломаной R для делимых пунктов (сортировка по убыванию a_i/b_i)
  2. Генерация множества S всех возможных распределений неделимых пунктов (2^M комбинаций)
  3. Выделение Парето-множества SP (оптимальные точки)
  4. Для каждой точки SP:
    • Построение смещённой ломаной R*
    • Проверка условий пропорциональности в вершинах
    • Геометрическая проверка на отрезках
